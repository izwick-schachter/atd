    module ATD::RequestHandlers

    This module is responsible for delegating http verb unique parsing methods. Currently doesn't do much.

    get() click to toggle source

    Processes get routes. Returns either the filename or plaintext output, and sends it back to ATD::App, where it is then sent to ATD::Renerers. The call could be shorter if you skiped the sending to ATD::App and just sent the ouput streight to ATD::Renderers.

    # File lib/atd.rb, line 136
def self.get
        all
        if !Path.paths[App.path_info].output.is_a?(Hash)
                return {:"content-type" => "text/plain", :content => Path.paths[App.path_info].output}
        else
                return Path.paths[App.path_info].output
        end
end

def all
        Path.paths[App.path_info].action.call unless Path.paths[App.path_info].action == nil
end
